metodo POST

	-password : stringa
	-nome : stringa

http://lucatedeschini2000.pythonanywhere.com/API/ListaProdotti
metodo POST

richiede in ingresso:
	-categoria : stringa

restituisce (leggi anche dopo)
	-idProdotto : int
	-tipologia : stringa
	-nome : stringa
	-descrizione : stringa
	-marca : stringa
	-prezzo : int

visto che da una ricerca possono uscire più risultati, restituisce un file json formattato in questo modo
{risultati:{1:{...},2:{...},3:...}}
quando lo converti in php dovresti riuscire ad accedere i dati così ma non ho provato
dati['risultati'][i]['marca']

dove i è un intero che va da 0 fino al massimo possibile (quindi dovresti fare un for) se da problemi dimmelo che dio è un po' così
